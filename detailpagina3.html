Linux Client-Serverproject

Beschrijving:

Voor deze opdracht heb ik een veilige en efficiënte Linux client-serveromgeving opgezet.
De focus lag op het configureren van partities, gebruikersbeheer, SSH-beveiliging, en het instellen van netwerkdiensten zoals NFS.
Daarnaast implementeerde ik back-upscripts en werd de server opgezet zonder grafische interface (headless), wat bijdraagt aan performance en veiligheid.

Doelstelling:

Het creëren van een stabiele, beheerbare en goed gedocumenteerde serveromgeving die eenvoudig opnieuw opgezet kan worden via configuratiescripts.

Mijn rol:

  • Installatie en configuratie van server en client
  • Automatisatie via shellscripts
  • Beveiligde SSH-toegang configureren
  • Partities en gebruikers correct beheren
  • Documentatie en rapport in PDF-formaat opstellen

Gebruikte tools en technieken:

  • Ubuntu Server & Client
  • Bash scripting
  • OpenSSH
  • NFS
  • rsync (voor back-up)
  • GParted (voor handmatige partitiecontrole)

Resultaat:

Het project leverde een volledig werkende serveromgeving op met correcte toegang per dienst,
geconfigureerde partities, geactiveerde en geteste NFS-shares, én een backupoplossing.
Alles werd getest en opgeleverd met een duidelijke technische documentatie.
